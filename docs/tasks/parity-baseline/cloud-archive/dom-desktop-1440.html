Cloud Provider

Set up a cloud storage provider to automatically archive dashcam footage.

How to connect your cloud storage:
  1. Install rclone on a computer with a web browser
  2. Open a terminal and run:
    rclone authorize "onedrive"
  3. Sign in and grant access in your browser
  4. Copy the token output and paste it below
Custom rclone backend

Connect to a NAS or any rclone-supported storage backend. Use the Form tab for guided setup, or the Paste tab if you already have an rclone.conf block from rclone config.

Required fields are marked with *. See the rclone docs for what each field means.

Paste a complete [remote] block from your existing ~/.config/rclone/rclone.conf. The remote name (in brackets) is ignored — TeslaUSB always stores it as teslausb.

Sync Settings
How sync works:
  • Automatic: Starts syncing whenever WiFi connects
  • Events: Syncs Sentry and Saved event folders by default
  • Recent (telemetry): Optionally syncs RecentClips files when the vehicle was moving (GPS/SEI data was recorded for the clip)
  • Oldest first: Within each folder, preserves the most at-risk clips first
  • Skip existing: Files already on cloud storage are never re-uploaded
  • Manual: Click “Sync Now” at any time

All camera angles for each event are uploaded together. You can continue browsing the web interface during sync.

Use the arrows to reorder. Top item is synced first.
