Tesla embeds GPS coordinates and telemetry data (speed, braking, steering) inside each dashcam video. The indexer always extracts this data from videos that carry GPS or SEI telemetry and builds a database of trips, routes, and driving events. This powers the map view — showing where you drove, plotting routes, and detecting events like hard braking or sharp turns.

TRIM

The read-only check runs e2fsck -nf against the mounted root filesystem. It makes no changes and is safe to run any time, but may report false positives because metadata is in flux on a read-write mount. Typically takes 30–90 s on a healthy SD card. The check also runs automatically in the background at most once every 7 days.

SD cards expose no wear telemetry (no SMART, no per-block checksums). Plan to replace the card every 12 months and keep cloud archive enabled so a card failure never costs you data.
